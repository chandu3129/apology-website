# 💕 Apology Website

A beautiful, romantic website to apologize to your girlfriend and show her how much she means to you.

## Features

✨ **Beautiful Design** - Gradient backgrounds with smooth animations  
💌 **Relationship Timeline** - Your journey together from 2023  
💕 **10 Reasons I Love You** - Personal reasons why she's special  
📝 **Memories Section** - Add and share beautiful memories  
🎯 **Heartfelt Messages** - Express your feelings  
📱 **Fully Responsive** - Works on mobile, tablet, and desktop  

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js + Express
- **Database**: SQLite
- **Deployment**: Vercel

## Local Development

1. Navigate to server folder:
```bash
cd server
npm install
npm start
```

2. Open http://localhost:3000 in your browser

## API Endpoints

- `GET /api/timeline` - Relationship timeline
- `GET /api/reasons` - Reasons why you love her
- `GET /api/messages` - Apology messages
- `GET /api/memories` - Stored memories
- `POST /api/memories` - Add new memory
